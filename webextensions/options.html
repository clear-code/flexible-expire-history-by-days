Expert and Scary Options

Differently from the previous version of this add-on, this version cannot disable expiration.

Disabling expiration is something to be done with extreme care, since the browser expiration algorithm has been built to provide the best performance.
Disabling Expiration can cause performance issues, especially in the location bar, and it's normally unnecessary.
If you really need a slow but infinite history, you can open about:config and create a new integer pref named places.history.expiration.max_pages with a value of 999999.
You can then setup your own expiration limit through this add-on.
